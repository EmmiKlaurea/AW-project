Accessibility Statement

Website: Emmi Tuominen Web Consulting

Publication date: 31st of July 2025

This accessibility statement was prepared on 31st of July 2025.

1. Accessibility Statement

This website aims to comply with the WCAG 2.1 guidelines at level AA as closely as possible. The site has been designed with accessibility in mind from a technical, visual, and content perspective. Web Content Accessibility Guidelines (WCAG) version 2.1
All devices are compatible with the website.

2. Accessible Features

  • Semantic HTML code: The elements comply with the accessibility requirements.
  • Screen reader compatibility: Alt texts are provided for images, form fields have clear labels, and ARIA attributes have been added where necessary.
  • Keyboard navigation: All features are accessible without using a mouse.
  • Color contrast: The colour palette meets the contrast requirements of at least WCAG 2.1 Level AA.
  • Responsiveness: The website is compatible with different devices and responsive to various screen sizes.
  • Zoom capability: Content remains readable and functional when zoomed in to 200%.
  • Clear language and structure: The content is written in simple, logical, and easy-to-understand language.

3. Known Accessibility Issues

  • This is a student project, so the scope of accessibility testing has been limited.
  • Screen reader testing has only been performed using NVDA.
  • Some elements may not have full ARIA roles or states defined.
  • Certain dynamic features, such as form error messages, may not yet be fully accessible to all assistive technologies.
  • Improvements will be made where possible.
  • When pressing number 1 on keyboard, Header 1 is not readable with a screen reader. Using the mouse, NVDA reads the heading.
  • Data privacy and Terms of use -pages lead to external page without a warning.

4. Accessibility Evaluation

  • Manual keyboard testing
  • Testing with the NVDA screen reader
  • Automated testing for responsiveness using the Axe DevTools extension.
  • Informal user testing with two individuals with visual impairments. Testing has been conducted with 5 people altogether to improve accessibility and usability. Most of the testing is made by content creater.

5. Feedback and Contact Information

If you encounter any accessibility issues on this website, you are welcome to provide feedback:

Email: emmi.tuominen@student.laurea.fi

Feedback will be reviewed within: as soon as possible.

6. Enforcement Procedure

Since this is a student project and not an official website of a public organization, there is no formal enforcement procedure. However, all feedback is taken seriously and will be considered in future development work. If this was an official website, there could be a possibility to contact an appropriate authority which is Traficom in Finland.
